Introduction


Giro3D (https://giro3d.org/) est une librairie permettant de faire du webmapping 3D.


Comme pour OpenLayer, il y a une liste d'exemples très complète : https://giro3d.org/latest/examples/index.html


Nous allons voir ici les bases de Giro3D

Giro3D

Installation


Giro3D s'installe avec NPM, le nom du package est @giro3d/giro3d

La commande est donc : npm install @giro3d/giro3d

Première application


Au coeur de Giro3D se trouve l'objet Instance. Il représente toute la "boite" Giro3D sur la page.


    import Instance from "@giro3d/giro3d/core/Instance.js";

    const instance = new Instance({
      crs: CoordinateSystem.epsg4326,
      target: "view",
      backgroundColor: null,
    });

Première application


Il est possible de définir sa propre projection


    import CoordinateSystem from "@giro3d/giro3d/core/geographic/CoordinateSystem.js";

    const crs2056 = CoordinateSystem.register(
      "EPSG:2056",
      "+proj=somerc +lat_0=46.95240555555556 +lon_0=7.439583333333333 +k_0=1 +x_0=2600000 +y_0=1200000 +ellps=bessel +towgs84=674.374,15.056,405.346,0,0,0,0 +units=m +no_defs",
    );
  

    const instance = new Instance({
      crs: crs2056,
      target: "view",
    });

Première application


Pour créer la map, il faut lui donner l'étendue à afficher


    import Extent from "@giro3d/giro3d/core/geographic/Extent.js";
    import Map from "@giro3d/giro3d/entities/Map.js";

    const extent = new Extent(
        crs2056, // Par exemple le CRS définit précédemment
        2496595, 2837805, 1067184, 1291771
    );

    const map = new Map({ extent });
    // Ajoute la map à l'instance. 
    // Tous les éléments (entités) de Giro3D doivent être ajoutés à l'instance.
    instance.add(map);

Pour charger une couche WMTS dans la map 


    import ColorLayer from "@giro3d/giro3d/core/layer/ColorLayer.js";
    import WmtsSource from "@giro3d/giro3d/sources/WmtsSource.js";

    WmtsSource.fromCapabilities(
      "https://wmts.geo.admin.ch/EPSG/3857/1.0.0/WMTSCapabilities.xml?lang=fr",
      {
          layer: "ch.swisstopo.swisssurface3d-reliefschattierung-multidirektional",
          matrixSet: "EPSG:2056",
      },
    ).then((source) =>  map.addLayer(new ColorLayer({ name: "wmts", source })))

Finalement, il faut positionner la vue

Il existe plusieurs manières de le faire, par exemple


    import { MapControls } from "three/examples/jsm/controls/MapControls.js";

    const center = extent.centerAsVector2();
    instance.view.camera.position.set(center.x, center.y - 1, 500000);

    const controls = new MapControls(instance.view.camera, instance.domElement);
    controls.target.set(center.x, center.y, 0);

    instance.view.setControls(controls);

Un MapControls est le contrôle permettant de bouger une carte en 3D.
