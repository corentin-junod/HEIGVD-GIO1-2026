# Géoinformatique opérationnelle : développement avancé d'outils (GIO1)

https://www.jmdt.ch/presentation/modules-et-ateliers/geoinformatique-operationnelle-developpement-avance-doutils

Le but de ce module est le renforcement des compétences en développement informatique, notamment au niveau de l’architecture du code, au niveau de l’optimisation du code et au niveau des outils qui sont utilisés pour le développement informatique.

## Objectifs

À la fin de ce cours, l’étudiant·e sera capable de :
- Concevoir une application web (HTML, CSS, JS)
- Utiliser les outils modernes de gestion de code (git, GitHub)
- Utiliser un framework web pour faciliter le développement (VueJS)
- Intégrer une librairie web cartographique (Leaflet, Openlayers, etc)
- Produire un code propre, structuré et organisé

## Prérequis 

Avoir suivit un cours SIG de base et avancé (par exemple HEIG-VD : SIG1, SIG2, SIG3, SIG4)

Informatique : 
- Algorithmique de base
- Développement HTML, Javascript, CSS
- Programmation orientée objet
- Outils de mapping web, comme OpenLayers ou Leaflet

## Évaluation

Le module est évalué en session d’examen, pendant le semestre à la fin du module :
- Projet de groupe, avec présentation (50% de la note finale)
- Oral individuel (50% de la note finale)

### Projet

Le projet consiste à développer, par groupe, une application web mettant en application les notions vue en cours.

Le projet devra :
- Se présenter sous le forme d'une application web, utilisant HTML, CSS et JavaScript
- Utiliser la librairie VueJS
- Utiliser au moins une librairie de mapping (leaflet, openlayers, cesiumJS ou Giro3D)
- Présenter des données issues d'une API externe (par exemple https://docs.geo.admin.ch/ ou similaire)
- Utiliser git pour la gestion du code et être hébergé sur GitHub
