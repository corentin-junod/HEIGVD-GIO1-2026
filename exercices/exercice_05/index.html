Exercice 05

Note : Cet exercice utilise les modules JavaScript. Pour que ceux-ci fonctionnent en local sur votre ordinateur, il vous faut un serveur web local. Vous pouvez démarrer un serveur web local en ouvrant un terminal et en vous plaçant dans le dossier des exercices, puis en tapant la commande : python3 -m http.server ou python -m http.server.
Dans votre navigateur, allez ensuite à la page 127.0.0.1:8000

Créez les modules :

  • Point.js
  • Polygon.js


Point.js déclare une classe "Point", qui contient les attributs suivants :

  • x, sa coordonnée X
  • y, sa coordonnée Y

Et qui contient les méthodes suivantes :

  • distanceTo, prenant en paramètre un autre point, et qui retourne la distance entre le point et cet autre point
  • translate, qui prend deux arguments, dx et dy, et qui déplace le point de dx selon l'axe X et de dy selon l'axe Y

Le module Point.js déclare également une constante de type Point, nommée ORIGIN, qui retourne le point (0;0)



Polygon.js déclare une classe "Polygon", qui contient l'attribut "points", qui est un tableau de Point (classe décrite ci-dessus).

Ces points décrivent les sommets d'un polygone, dans l'ordre, et on suppose que le périmètre du polygone ne s'intersecte pas avec lui-même.

La classe Polygon contient les méthodes suivantes :

  • addPoint, qui ajoute un sommet, de type Point à la suite des sommets du polygone
  • getArea, qui retourne l'aire du polygone. Vous pouvez utiliser la formule du lacet pour calculer cela (https://en.wikipedia.org/wiki/Shoelace_formula)
  • getPerimeter, qui retourne le périmètre du polygone
  • translate, qui prend les paramètres dx et dy, et déplace le polygone de dx en X et de dy en Y
  • getCenter, qui retourne le centre du polygone, définit comme la moyenne des abscisses et des ordonnées de ses sommets

Le module Polygon.js déclare également les constantes :

    ORIGIN_SQUARE, qui est le Polygon qui décrit un carré dont les sommets sont (0;0), (0;1), (1;1) et (1;0)
    ORIGIN_TRIANGLE, qui est le Polygon qui décrit un triangle dont les sommets sont (0;0), (0;1) et (1;0)



Le fichier mains.js contient des tests, servant aussi d'exemples d'utilisation de toutes les fonctions décrites ci-dessus. Ce fichier, en s'exécutant, ne doit retourner aucune erreur.
