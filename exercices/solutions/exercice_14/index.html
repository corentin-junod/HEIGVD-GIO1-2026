Exercice 14

L'objectif de cet exercice est des développé une application en Vue.js comportant deux composants.

  • Le premier composant est une carte OpenLayers, affichant la même couche qu'a l'exercice 10. (couche ch.swisstopo.swisssurface3d-reliefschattierung-multidirektional du serveur https://wmts.geo.admin.ch/EPSG/3857/1.0.0/WMTSCapabilities.xml?lang=fr)
  • Le deuxième composant est une carte Giro3D, affichant cette même couche

Références :

  • Pour créer automatiquement un nouveau projet avec Vue.js, utilisez la commande npm create vue@latest.
    La ligne de commande vous demandera alors
    • un nom pour votre projet, par exemple "ex_14"
    • si vous souhaitez utiliser TypeScript (sélectionnez non)
    • si vous souhaitez d'autres fonctions (non, appuyiez simplement sur "enter")
    • si vous souhaitez d'autres fonctions expérimentales (non, appuyiez simplement sur "enter")
    • si vous voulez ne pas avoir de code d'exemple dans votre projet (sélectionnez oui)
    Puis finalement placez la ligne de commande dans le projet nouvellement créé, et installez Openlayers et Giro3D : npm i ol @giro3d/giro3d
    Vous pouvez lancer le serveur de développement local avec npm run dev
