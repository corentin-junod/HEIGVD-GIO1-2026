Exercice 12

Note : Cet exercice utilise les modules JavaScript. Pour que ceux-ci fonctionnent en local sur votre ordinateur, il vous faut un serveur web local. Vous pouvez démarrer un serveur web local en ouvrant un terminal et en vous plaçant dans le dossier des exercices, puis en tapant la commande : python3 -m http.server ou python -m http.server.
Dans votre navigateur, allez ensuite à la page 127.0.0.1:8000

En relisant votre exercice 05, vous remarquez que les classes Point et Polygon ont plein d'effets de bord.

Réécrivez ces classes de manière à les rendre immuables. C'est à dire qu'un Point ou un Polygon, une fois créé, ne doit pas pouvoir être modifié.

Dans la classe Point, ajouter les méthodes getX et getY qui permettent de lire les attributs x et y.

Dans la classe Polygon, ajoutez la méthode getPoints, qui retourne les sommets du Polygon sous forme de Point.

Les constantes ORIGIN, ORIGIN_SQUARE et ORIGIN_TRIANGLE doivent aussi être immuables.
