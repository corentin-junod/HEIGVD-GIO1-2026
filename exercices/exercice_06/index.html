Exercice 06

Note : Cet exercice utilise les modules JavaScript. Pour que ceux-ci fonctionnent en local sur votre ordinateur, il vous faut un serveur web local. Vous pouvez démarrer un serveur web local en ouvrant un terminal et en vous plaçant dans le dossier des exercices, puis en tapant la commande : python3 -m http.server ou python -m http.server.
Dans votre navigateur, allez ensuite à la page 127.0.0.1:8000

Cet exercice se veut un peu plus libre que les autres. Si vous ne savez pas comment faire quelque chose, n'hésitez pas à chercher sur internet.


L'objectif est de créer plusieurs sliders interactifs, dont la valeur affichée sous forme de texte se met à jour automatiquement.


Dans cet exercice, créez les classes suivantes, chacune dans son module :

  • BaseSlider
    • Cette classe crée un slider sur la page avec un texte indiquant sa valeur
    • Le constructeur prend les arguments suivants
      • parent : l'élément HTML parent, dans lequel placer le slider
      • label : le texte accompagnant le slider
      • min : la valeur minimale du slier
      • max : la valeur maximale du slider
    • Le slider sera composé de :
      • Un élément div qui servira de conteneur
      • Un élément span pour le label
      • Un élément span pour afficher la valeur du slider
      • Un élément input de type "range" pour faire le sélecteur de valeur.
        Il est possible de définir la propriété "type", le minimum, le maximum et la valeur d'un input via ses attributs :
        • "type" pour le type
        • "min" pour le minimum
        • "max" pour le maximum
        • "value" pour la valeur actuelle
    • Voir les fonctions suivantes pour interagir avec les éléments d'une page web en JavaScript:
        document.createElement pour créer un élément HTML
        element.append pour ajouter un élément à un autre
        element.addEventListener pour ajouter un événement à un élément HTML. L'événement du type "input" permet d'appeler une fonction à chaque changement de valeur d'un input
        element.textContent pour définir le contenu textuel d'un élément
  • PercentageSlider, qui sera un BaseSlider allant toujours de 0 à 100, et dont le label affichera sa valeur en pourcent. Par exemple "53 %"
  • ExponentialSlider, qui sera un BaseSlider prenant un paramètre supplémentaire, nommé "exponent", et qui affichera dans son label la valeur du slider élevé à la puissance de l'exposant.
  • ColorSelector, qui sera un ensemble de 3 BaseSlider, respectivement pour le rouge, le vert et le bleu d'une couleur RGB.
    Sa méthode "setOnChange" prend en paramètre une fonction qui sera appelée à chaque fois que la couleur indiquée par ses sliders change.
    Son utilisation est montrée dans le fichier main.js. La couleur est renvoyée sous la forme "rgb(20,30,40)"
